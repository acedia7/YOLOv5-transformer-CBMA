# YOLOv5-transformer-CBMA
是第一次对yolo的学习和尝试

使用了yolov5的源码，并在源码上做了一点修改，加入了transformer和CBMA模块，增强了模型的鲁棒性。

该模型训练使用了 VOC2012的数据集（数据集并没有在仓库中，还需自行下载）
